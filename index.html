Cards de perguntas

O projeto consiste em uma aplicação web interativa, desenvolvida com HTML, CSS, e JavaScript, que ajuda no estudo de novas coisas, curiosidades recentes, antigas, etc.

O HTML estrutura o conteúdo da página, dividindo-o em seções como a tela inicial, caixa de perguntas, alternativas de resposta, e o resultado final baseado nas escolhas do jogador. O CSS é utilizado para estilizar a página, empregando um esquema de cores chamativas e layout responsivo para melhorar a experiência do usuário. JavaScript é usado para adicionar interatividade ao jogo, manipulando o DOM para animar os cards e ajudar na interação.

Os scripts JavaScript incluem módulos para gerar cards animados, definir as perguntas e as lógicas de jogo, como iniciar o jogo, apresentar perguntas e alternativas, processar as escolhas dos jogadores, e exibir o resultado final. Este design modular facilita a manutenção e a expansão do jogo. A importância das ferramentas utilizadas (HTML, CSS, JavaScript) reside na criação de uma experiência interativa que não apenas entretém, mas também educa os jogadores sobre coisas que vemos em nossa vida, mas não lembramos.
